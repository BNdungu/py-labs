## Your Task

### Part A — Multiplication Table
Ask the user for a number, then print its multiplication table from 1 to 10.

### Example output
```
Enter a number: 3
3 x 1 = 3
3 x 2 = 6
3 x 3 = 9
3 x 4 = 12
3 x 5 = 15
3 x 6 = 18
3 x 7 = 21
3 x 8 = 24
3 x 9 = 27
3 x 10 = 30
```

### Part B — Sum of Numbers
Ask the user how many numbers they want to add. Then ask them to enter each number one by one. At the end, print the total.

### Example output
```
How many numbers? 3
Enter number 1: 10
Enter number 2: 5
Enter number 3: 8
Total: 23
```

---

### Part C — Password Retry
Set a secret password (e.g. `"python123"`). Keep asking the user to enter the password until they get it right. Each wrong attempt prints `"Incorrect, try again."`. When correct, print `"Access granted!"`.

### Example output
```
Enter password: hello
Incorrect, try again.
Enter password: python123
Access granted!
```
